Python Project *sympy_random_matrix*
-----------------------------------------------------------------------


Introduction
------------

To import the project simply type

.. code-block:: python

    >>> import sympy_random_matrix

after installation.


Install
-------

The latest stable version can always be installed or updated via pip:

.. code-block:: bash

    $ pip install sympy_random_matrix
